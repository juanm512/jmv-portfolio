## El desafío

Muchas personas luchan con el seguimiento de sus gastos y la planificación financiera. Las apps existentes son complejas o no se integran bien con bancos locales.

## La solución

Desarrollé una app móvil intuitiva que incluye:

- Conexión segura con cuentas bancarias via Plaid
- Categorización automática de transacciones
- Presupuestos adaptativos basados en hábitos
- Metas de ahorro con seguimiento visual
- Recordatorios de pagos y suscripciones

## Tecnologías utilizadas

- **Mobile**: React Native con Expo
- **Backend**: Firebase Functions y Firestore
- **Integraciones**: Plaid API para datos bancarios
- **Visualización**: Chart.js para gráficos interactivos

## Resultados

- 10,000+ descargas en 6 meses
- Usuarios reportan ahorro promedio de 20% mensual
- 4.6 estrellas en App Store y Play Store
